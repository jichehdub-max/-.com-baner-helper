Тест Multipass Rendering для Shadertoy

Инструкция:
  • Вставьте код шейдера из Shadertoy (только Image shader)
  • Для шейдеров с Buffer A/B/C/D - используйте только код из вкладки "Image"
  • Плагин автоматически создаст ping-pong буферы для temporal reprojection
  • iChannel0 будет содержать предыдущий кадр (для motion blur, TAA, и т.д.)
