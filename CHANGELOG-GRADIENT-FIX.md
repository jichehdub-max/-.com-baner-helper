# Changelog: Исправление градиента и обновление шейдеров

## Commit: 0dcbde5

### Исправления

#### 1. Плавная анимация градиента
**Было:**
- 2 ключевых кадра (0%, 50%, 100%)
- Резкие переходы
- 10 секунд с `ease`

**Стало:**
- 4 ключевых кадра (0%, 33%, 66%, 100%)
- Плавные переходы между 3 оттенками
- 15 секунд с `ease-in-out`
- Более естественная анимация

**Код:**
```javascript
@keyframes customGradient {
  0% {
    background: linear-gradient(135deg, bg1 0%, bg2 50%, bg1 100%);
  }
  33% {
    background: linear-gradient(135deg, bg2 0%, bg3 50%, bg2 100%);
  }
  66% {
    background: linear-gradient(135deg, bg3 0%, bg1 50%, bg3 100%);
  }
  100% {
    background: linear-gradient(135deg, bg1 0%, bg2 50%, bg1 100%);
  }
}
```

#### 2. Обновление существующего стиля
- Добавлена проверка существования стиля градиента
- Если стиль уже есть - обновляется, а не создаётся новый
- Предотвращает дублирование `<style>` элементов
